Um player de música web moderno, interativo e com design estilo *neon/cyberpunk*. Construído com React e estilizado com Tailwind CSS, este projeto foca em oferecer uma experiência visual imersiva, combinando faixas de áudio com artes em vídeo animadas e um equalizador dinâmico.

## ✨ Funcionalidades

* **Sincronia de Mídia:** O vídeo de capa (artwork) pausa e reproduz em perfeita sincronia com a faixa de áudio atual.
* **Artwork em Vídeo Aleatória:** A cada troca de faixa (Próxima/Anterior), um novo vídeo de fundo é sorteado aleatoriamente de uma lista predefinida.
* **Equalizador Animado:** Barras de equalização que ganham vida e se movimentam de forma orgânica (com *delays* variados) quando a música está tocando.
* **Tempo "Infinito":** Interface estilizada para transmissões ao vivo ou loops de lofi, exibindo o contador de tempo atualizado junto com um símbolo de infinito (`∞`).
* **Design Responsivo & Glassmorphism:** Interface fluida construída com Tailwind CSS, apresentando efeitos de desfoque, sombras brilhantes e bordas arredondadas.

## 🚀 Tecnologias Utilizadas

* **React.js:** Biblioteca principal para renderização e gerenciamento de estado (`useState`, `useRef`, `useEffect`).
* **Tailwind CSS:** Framework utilitário para estilização rápida, responsividade e efeitos visuais (sombras, blur).
* **HTML5 Audio/Video API:** Manipulação nativa de mídia via referências do React para controle total de *play/pause*.
